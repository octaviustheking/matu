# Matu

Matu is a lightweight game engine designed to build 2D web games. It is inspired by editors such as Godot and Unity, while aiming for the simplicity of Scratch. 

## Important: check [HERE](docs.md) for documentation!

## Features

 - Node system consisting of Groups, Objects, Sprites, Audio, and Scripts
 - Scripting functionality
 - Asset uploading
 - Viewport testing
 - Saving and loading
 - Exporting into a web playable
